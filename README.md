## La Certificación CEHPC

La certificación **Ethical Hacking Professional Certificate** de CertiProf valida las habilidades y conocimientos en:

- Fundamentos de hacking ético
- Técnicas de penetration testing
- Identificación de vulnerabilidades
- Evaluación de seguridad en sistemas y redes
- Metodologías de Ciberseguridad
