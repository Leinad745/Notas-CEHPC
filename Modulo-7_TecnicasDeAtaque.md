# Modulo 7 - Tecnicas de Ataque

## Tipos de ataque

### Malware

Software malicioso, intrusio e intencionalmente desagradable que intenta invadir, dañar o deshabilitar ordenadores, sistemas informaticos, redes, tabletas y dispositivos moviles, a menudo asumiendo el control parcial de las operaciones de un dispositivo.

### Spoofing

El spoofing usurpa la identidad electronica para ocultar la propia identidad y cometer delitos informaticos. Existen 3 tipos:

- De correo electronico: virus enviados desde  direcciones de correo electronico existentes, con el fin de engañar al destinatario.
- De IP: envia paquetes IP desde una direccion IP no asignada al ordenador que los envia.
- Smart-Spoofing IP: usa cualquier aplicacion cliente usurpando la IP. Esto evita las reglas de seguridad de la red.

### MITM

Los ataques MAN IN THE MIDDLE ocurren cuando una comunicacion entre dos sistemas es interceptada por una entidad externa, funcionando como un intermedio entre ambos lados.

### DDos

Inhabilita el uso de un sistema bloqueando el servicio destinado. los servidores web poseen la capacidad de resolver un numero determinado de peticiones o conexiones antes de comenzar a ralentizarse o a empezar a ignorar respuestas y bloquearse.

### PiggyBacking

Usa conexiones inalambricas para acceder a una conexion a internet sin autorizacion. Obtiene un acceso libre a la red que a menudo se aprovecha para intentar actividades maliciosas como violacion de datos y difusion de malware.

### Inyeccion SQL

Falla de codificacion que posibilita por medio de un input cualquiera la manipulacion de una consulta SQL, podiendo conseguir datos, accesos no autorizados y distinta informacion confidencial de la base de datos.
